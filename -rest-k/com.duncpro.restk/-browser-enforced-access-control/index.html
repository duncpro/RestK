val allowedMethods: Set<HttpMethod>
Content copied to clipboard

the set of all methods which an origin has permission to access. If empty, then the origin is considered forbidden, and no Access-Control-Allow-Origin, or Access-Control-Allow-Methods header will be attached to the response. If the set contains at least one method, then both of the aforementioned headers will be automatically included in the response.
