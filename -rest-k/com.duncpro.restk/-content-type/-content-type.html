Creates a new ContentType instance with an arbitrary mime-type, mime-subtype, and charset. It is possible to create ContentTypes which will result in an error when accessing ContentType.charset. Therefore, the caller should take care to ensure the given charset is supported on the current platform.

To achieve case-insensitivity, the provided string arguments will be converted to lowercase text.
